About the role

Datazip is looking for a Software Engineer Intern to join our backend team! We're building a next-generation composable data lakehouse platform that helps organizations streamline their data engineering workflows and unlock insights from their data.

As a Backend Engineering Intern, you'll work directly with our engineering team to build scalable backend services and APIs that power our data platform. This is a hands-on role where you'll learn modern backend development practices and gain experience working with big data technologies.

What you'll do

  • Develop and maintain RESTful APIs using Python, FastAPI, and SQL
  • Implement data processing pipelines using distributed computing frameworks
  • Write unit tests and integration tests for backend services
  • Collaborate with frontend engineers to integrate backend APIs
  • Participate in code reviews and technical discussions
  • Learn about cloud infrastructure and deployment pipelines
